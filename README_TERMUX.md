# تشغيل المشروع على أندرويد (Termux)

لضمان عمل المشروع بدون مشاكل أذونات (EACCES) أو أخطاء الروابط الرمزية (Symlinks) على أندرويد، اتبع الخطوات التالية:

## 1. المتطلبات
تأكد من تثبيت Node.js في Termux:
```bash
pkg update && pkg upgrade
pkg install nodejs
```

## 2. التثبيت الصحيح
إذا كنت تقوم بتشغيل المشروع من ذاكرة الهاتف الداخلية (`/sdcard` أو `Downloads`)، يجب استخدام الأمر التالي لتجنب مشاكل الأذونات:
```bash
npm install --no-bin-links
```
أو ببساطة استخدم السكربت المجهز:
```bash
npm run termux-fix
```

## 3. التشغيل
لتشغيل المشروع في وضع التطوير:
```bash
npm run dev
```

## 4. ملاحظات هامة
- تم تعطيل الروابط الرمزية (bin-links) في ملف `.npmrc` لضمان التوافق مع أنظمة ملفات أندرويد (FAT32/exFAT).
- تم تحديث جميع الحزم (Dependencies) لأحدث الإصدارات المستقرة لتجنب تحذيرات `deprecated`.
- إذا واجهت مشكلة في `better-sqlite3` على Termux، قد تحتاج لتثبيت أدوات البناء: `pkg install build-essential python`.
